# 연산자
- 산술연산자
```python
+ - * / // ** %
```
```python
a = 10 + 3      # 13
b = 10 - 3      # 7
c = 10 * 3      # 30
d = 10 / 3      # 3.3333333333333335 (float)
e = 10 // 3     # 3  (몫, 버림)
f = 2 ** 4      # 16
g = 10 % 3      # 1
```
- 자주하는 실수 방지
    - // 와 / 를 혼동하지 않기: / → 실수, // → 몫(버림)
 

- 논리연산자
```python
and or
```
